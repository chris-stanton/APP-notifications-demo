Notifications Demo
A simple application that allows Developers to view all of the most popular Javascript 3rd party alert libraries at one location. The application's framework is built from HTML5, CSS3, Javascript, Flex Box and Waypoints.js. The only 3rd party plugins are the alert libraries intended for this demonstration. There is NO JQUERY in this application.
