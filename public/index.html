Notifications Demo

A simple application that will allow Developers to view all of the most popular Javascript 3rd party alert libraries at one location. The apllication's framework is build from HTML5, CSS3, Javascript and Flex Box. The only 3rd party plugins are the alert libraries intended for this demonstration. There is NO JQUERY in this application.
